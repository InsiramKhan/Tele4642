Project Overview

What is IntelliHome SDN?

IntelliHome SDN is a smart-home network framework built on Software Defined Networking (SDN), combining manufacturer-specific access control policies, real-time machine learning-based threat detection, and flow trust ranking. The system dynamically adapts to changing traffic behavior, ensuring both performance and security in residential IoT environments.

Motivation

As smart homes become more device-rich, traditional networking approaches fall short in enforcing access boundaries or detecting malicious behavior. Common routers lack visibility into device intent, often exposing home networks to IoT-based threats like botnets, data leaks, or unauthorized lateral access.

Key Objectives

  • Policy Enforcement: Parse and enforce Manufacturer Usage Description (MUD) JSON profiles to limit device communication strictly to intended targets.
  • Threat Detection: Train and integrate a real-time machine learning model (Random Forest) to flag and react to suspicious flows.
  • Trust-based Flow Prioritization: Use PageRank to score flows based on connection patterns, allowing more adaptive control over traffic handling.

Stack Summary

The project uses Mininet for emulation, Ryu for SDN control, and Python-based modules for MUD parsing, ML inference (via scikit-learn), and flow scoring (networkx).
